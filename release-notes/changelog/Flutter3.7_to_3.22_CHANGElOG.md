# Changelog: Flutter 3.7 to Flutter 3.22

From Flutter 3.7 to 3.22, the version updates have brought many improvements. These include performance optimizations, new features, enhancements in security and stability, as well as improvements in development tools. In addition, there has been progress in the expansion of the ecosystem. Developers need to pay attention to the changes that require adaptation.
